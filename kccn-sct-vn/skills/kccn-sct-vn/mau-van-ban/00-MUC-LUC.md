## Bản đồ tình huống → mẫu

- **DN muốn làm chủ đầu tư CCN** → 01.1 + 01.2 (DN lập) → 05.1 (SCT hướng dẫn).
- **UBND xã trình thành lập CCN** → 01.3 → SCT: 02.1 (lấy ý kiến) → 02.2 (thẩm định) → 02.3 (tờ trình) → 02.4/06-Mẫu 02 (dự thảo QĐ).
- **Có nhiều nhà đầu tư/cần chấm điểm** → 03.1 (lập HĐ) → 03.2 (tiêu chí) → 03.3 (phiếu) → 03.4 (biên bản) → 03.5 (báo cáo) → QĐ thành lập.
- **Hỏi đáp nghiệp vụ, hướng dẫn** → 05.1/05.2 + reference 10.
- **Hồ sơ thiếu** → 05.3.
- **Điều chỉnh/bãi bỏ QĐ** → 01.4 + 02.2/02.3 (điều chỉnh) + reference 11 mục D.
- **Báo cáo định kỳ/chuyên đề/nhanh** → 04.1/04.2/04.3.
- **Ban hành/sửa Quy chế CCN tỉnh, Quy chế dịch vụ tiện ích** → 06-Mẫu 03/Mẫu 04.
