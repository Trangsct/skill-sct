# 14 - Báo cáo cấp trên và thống kê

Hướng dẫn lập các loại báo cáo của Sở Công Thương về lĩnh vực vận chuyển HHNH. Soạn theo skill `vbhc-vn` (mẫu báo cáo, ký hiệu BC-SCT, ký SCT-CN, Lưu VT, CN). Tuyệt đối không bịa số liệu, số/ngày văn bản trích dẫn.

## 1. Các loại báo cáo và đầu mối nhận

| Loại báo cáo | Khi nào | Gửi ai (điển hình) |
|---|---|---|
| **Định kỳ** (tháng/quý/6 tháng/năm) về kết quả quản lý nhà nước lĩnh vực HHNH, ATVSLĐ | Theo lịch báo cáo của UBND tỉnh, Bộ Công Thương | UBND tỉnh; Bộ Công Thương (Cục Kỹ thuật an toàn và Môi trường công nghiệp/Cục Hóa chất theo phân công); cơ quan yêu cầu |
| **Chuyên đề / theo yêu cầu** (ví dụ kết quả triển khai TT 26/2026, kết quả cấp phép HHNH, kết quả kiểm tra) | Khi có văn bản chỉ đạo, đề nghị | Cơ quan chỉ đạo/đề nghị |
| **Đột xuất** sau sự cố vận chuyển HHNH | Ngay khi xảy ra sự cố/theo yêu cầu | UBND tỉnh; cơ quan liên quan |
| **Báo cáo thống kê số liệu** (số giấy phép cấp/điều chỉnh/cấp lại/thu hồi; số cuộc kiểm tra; số vụ xử lý) | Kèm báo cáo định kỳ hoặc theo biểu mẫu yêu cầu | Theo biểu mẫu của cơ quan yêu cầu |

CẢNH BÁO: kỳ hạn, đầu mối nhận và biểu mẫu báo cáo do văn bản chỉ đạo cụ thể quy định; **xác minh đúng văn bản yêu cầu báo cáo (số, ngày, cơ quan ban hành, biểu mẫu kèm theo) trước khi soạn** — không tự đặt kỳ hạn/đầu mối.

## 2. Bộ chỉ tiêu thống kê chuẩn (dùng cho mọi báo cáo định kỳ)

Lập bảng số liệu kỳ báo cáo, đối chiếu sổ theo dõi của Phòng QLCN:

| 1 | Số hồ sơ tiếp nhận | | | |
| 2 | Số Giấy phép cấp mới | | | theo loại 1/2/3/4/5/8/9 |
| 3 | Số Giấy phép cấp điều chỉnh | | | |
| 4 | Số Giấy phép cấp lại (mất, hỏng) | | | |
| 5 | Số hồ sơ yêu cầu bổ sung | | | |
| 6 | Số hồ sơ từ chối (nêu lý do chính) | | | |
| 7 | Số Giấy phép thu hồi | | | theo điểm vi phạm Điều 17 |
| 8 | Số trường hợp hướng dẫn/miễn cấp phép | | | |
| 9 | Số cuộc kiểm tra chuyên ngành | | | theo kế hoạch/đột xuất |
| 10 | Số vi phạm phát hiện | | | |

Khi chưa có số liệu thực: để trống/ghi "đang tổng hợp", **không điền số ước đoán**.

## 3. Khung BÁO CÁO ĐỊNH KỲ (cấu trúc nội dung)

```
BÁO CÁO
Kết quả quản lý nhà nước về vận chuyển hàng hóa nguy hiểm
(Kỳ: ......; thực hiện theo Văn bản số .../... ngày ... của ...)

I. CÔNG TÁC CHỈ ĐẠO, TRIỂN KHAI
- Văn bản đã ban hành/tham mưu (liệt kê đúng số, ngày - không bịa).
- Công tác tuyên truyền, hướng dẫn doanh nghiệp.

II. KẾT QUẢ THỰC HIỆN
1. Cấp phép (kèm bảng chỉ tiêu mục 2).
2. Kiểm tra chuyên ngành (số cuộc, đối tượng, kết quả).
3. Xử lý vi phạm (Sở xử phạt; chuyển cơ quan khác; thu hồi giấy phép).
4. Phối hợp xử lý sự cố (nếu có).

III. ĐÁNH GIÁ
- Kết quả đạt được; tồn tại, hạn chế; nguyên nhân.

IV. KHÓ KHĂN, VƯỚNG MẮC VÀ KIẾN NGHỊ
- Vướng mắc pháp lý (vd: căn cứ ủy quyền, phân định thẩm quyền loại 5/8, người áp tải...).
- Kiến nghị cơ quan có thẩm quyền.

V. PHƯƠNG HƯỚNG KỲ TỚI

Nơi nhận; quyền hạn, chức vụ người ký (PGĐ phụ trách - SCT-CN); Lưu: VT, CN(tên).
```

## 4. Khung BÁO CÁO ĐỘT XUẤT (sự cố vận chuyển HHNH)

```
BÁO CÁO NHANH
về sự cố trong vận chuyển hàng hóa nguy hiểm

1. Thời gian, địa điểm xảy ra sự cố.
2. Tổ chức/cá nhân, phương tiện, loại/khối lượng HHNH liên quan.
3. Diễn biến, thiệt hại ban đầu (người, tài sản, môi trường).
4. Tình trạng Giấy phép vận chuyển (có/không, còn hiệu lực).
5. Công tác ứng cứu, phối hợp (UBND cấp xã, PCCC, CSGT, y tế...).
6. Đánh giá nguyên nhân ban đầu; đề xuất xử lý.
7. Kiến nghị.
```

## 5. Lưu ý nghiệp vụ khi lập báo cáo

1. Mỗi số liệu phải có nguồn (sổ theo dõi, hồ sơ lưu); **không suy đoán, không làm tròn cho đẹp**.
2. Trích dẫn văn bản (số, ngày, cơ quan) phải đúng tuyệt đối; thiếu nguồn thì viết chung và ghi chú đề nghị xác minh.
3. Phân tách rõ "Sở xử phạt" và "chuyển cơ quan khác xử lý" để không phản ánh sai thẩm quyền (xem plugin xử phạt riêng `xlvphc-sct-vn`).
4. Bám đúng đề cương/biểu mẫu của văn bản yêu cầu báo cáo nếu có; nếu không có đề cương, dùng khung tại mục 3/4.
5. Báo cáo do PGĐ phụ trách lĩnh vực ký (SCT-CN); báo cáo quan trọng/Tờ trình UBND tỉnh do Giám đốc Sở ký - xác định đúng người ký trước khi hoàn thiện.
